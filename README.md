# Instituto Caminhos do Amanhã

## Descrição do Projeto
O **Instituto Caminhos do Amanhã** é um site institucional de uma ONG fictícia que apoia famílias em situação de vulnerabilidade por meio de projetos de alimentação, reforço escolar e atendimento psicológico.  
O projeto foi desenvolvido como atividade acadêmica, seguindo boas práticas de HTML5, CSS3 e JavaScript.

│

## Funcionalidades
- Estrutura de páginas **HTML5 semânticas** (index, projetos, cadastro, contato).  
- Layout responsivo usando **CSS3**, **Flexbox** e **Grid**.  
- Sistema de navegação com menu ativo e links entre páginas.  
- Formulário de cadastro de voluntário com campos obrigatórios.  
- Sistema básico de interatividade com **JavaScript** (validação simples de formulário).  
- Componentes visuais: cards, botões com estados, badges.  

## Tecnologias Utilizadas
- **HTML5** – marcação semântica e acessível.  
- **CSS3** – estilos, grid, flexbox, variáveis CSS e responsividade.  
- **JavaScript** – manipulação básica do DOM e validação de formulários.
